Carpeta para tus fotos

Coloca aquí todas las imágenes que quieras mostrar en el álbum.

Formatos soportados:
- .jpg / .jpeg
- .png
- .gif
- .webp

Para usar estas fotos:
1. Coloca tus imágenes en esta carpeta
2. Edita script.js y reemplaza el array 'photos' con las rutas de tus imágenes
3. Ejemplo: 'photos/mi-foto-1.jpg', 'photos/mi-foto-2.jpg', etc.

Nota: Para usar fotos locales necesitarás ejecutar el proyecto desde un servidor web local.
